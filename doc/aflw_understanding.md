# AFLW数据集

野外注解面部标志（AFLW）提供了从网络收集的大量注释面部图像，表现出各种各样的外观（例如姿势，表情，种族，年龄，性别）以及一般成像和环境条件。每张图片总共约25k张标注了最多21个地标。
